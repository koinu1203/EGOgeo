Frontend-only instructions for this folder:

- Follow `AGENTS.md` as the source of truth.
- Use Vue 3 + TypeScript, PrimeVue v4, Pinia, and Tailwind v4 conventions already defined in the project.
- Use PrimeVue components for supported UI patterns; do not use PrimeReact.
- PrimeVue styling must use variables/tokens in a reusable shared SCSS base layer.
- Do not use PrimeVue `pt` for styles.
- For non-PrimeVue elements, use Tailwind utility classes as needed.
- Keep all code-facing text in English.
- Use moderate validation: run type-check/build for meaningful code changes and report results.
- Keep changes frontend-scoped; do not include backend modifications unless explicitly requested.
